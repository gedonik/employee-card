
# Employees-card is a Vue application.

Vue app demonstration with bootstrap library.


## Installation

Clone my project, after type in terminal:

```bash
  npm install 
  npm run serve
```

## Tech Stack

HTML, CSS, SCSS, Vue, Bootstrap, Bootstrap-icons.
